# Maquette & Conception

Ce dossier contient les maquettes HTML et les références design du projet.

## Structure du dossier

L'organisation suit une logique fonctionnelle numérotée :

- **`01_structure-layouts/`** : Mises en page globales (Grilles, Wrappers).
- **`02_navigation/`** : Menus, Sidebars, Breadcrumbs.
- **`03_contenu-cards/`** : Affichage de données (Tables, Cards, Listes).
- **`04_recherche-filtres/`** : Champs de recherche, filtres avancés, pagination.
- **`05_formulaires-feedback/`** : Modales, Toasts, Uploads, Inputs complexes.
