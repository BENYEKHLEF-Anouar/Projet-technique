# Graphic Charter & Color System

## Primary Colors
- **Primary Base**: `teal-500` (Used for actions, active states)
- **Primary Text**: `primary` (Semantic token)

## Semantic Colors implementation
The following tokens should be mapped in the CSS configuration:

### Core
- `primary`: Main brand color.
- `secondary`: Secondary actions (often gray or muted).

### Functional
- `red-500`: Error states / Destructive actions.
- `yellow-500`: Warning states / Drafts.

### UI Layers
- `surface`: Base background (e.g., white).
- `overlay`: Modal or dropdown backgrounds.
- `layer`: Nested backgrounds (e.g., inputs, card headers).

### Typography
- `foreground`: Main text color.
- `muted-foreground`: Secondary text color.
- `bplain`: (To be defined - likely Black/Plain text)
