# Structure du Projet

Voici la structure des branches et dossiers du dépôt :

### Branches Principales
- **main** : Version finale et stable de l'application.
- **develop** : Branche de développement principale.

### Branches de Versioning (Sprints)
- **public** (v1) : Interface publique (Recherche, Filtre, Pagination).
- **admin** (v2) : Interface d'administration (CRUD, Modales).
- **gates** (v3) : Gestion des autorisations via les Gates.
- **spa-ajax** (v4) : Implémentation du mode SPA avec AJAX.
- **spa-alpine** (v5) : Refactorisation et optimisation avec Alpine.js.
- **spatie** (v6) : Gestion avancée des rôles et permissions avec Spatie.
- **api** (v7) : Développement de l'API REST.
- **mobile** (v8) : Branche dédiée à l'intégration mobile.

### Branches de Travail & Live Coding
- **prototype-ajax** : Code de démonstration initial utilisant AJAX.
- **live-coding-ajax-start** : Point de départ pour la session de live coding pour AJAX.
- **live-coding-ajax** : Résultat de la session de live coding pour AJAX.
